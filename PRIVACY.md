# Academate 隐私政策

更新日期：2026-06-03

Academate（以下简称「本应用」）是一款面向科研人员的学术论文管理与 AI 辅助阅读工具，由个人开发者运营，免费提供、非经营性质。本《隐私政策》说明我们如何收集、使用、存储和保护您的个人信息。您注册、点击「同意」或继续使用本应用，即表示您已阅读并接受本政策。

## 一、我们收集的信息

1. **账户信息**：您的姓名、邮箱、所在机构、注册邀请码，以及登录令牌。密码仅以加密哈希形式保存，我们**不存储明文密码**。
2. **您产生的内容**：您上传的论文文件、与 AI 的对话内容、收藏与阅读记录、偏好设置（主题/语言/字号/模型选择），以及您通过「意见反馈」提交的文字与图片。
3. **使用信息**：使用统计与阅读记录，以及为保障服务运行所必需的网络连接信息。

本应用仅申请「网络访问」权限，**不收集定位、通讯录、相册等敏感信息**。

## 二、信息的使用

我们使用上述信息用于：提供论文追踪与个性化推荐、AI 对话与阅读、保存您的文库与偏好、处理您的反馈、保障服务安全与正常运行。

## 三、存储与跨境传输

本应用的服务器位于中国大陆境内（www.aiforecon.cn）。无论您身处何地，使用本应用即意味着您的相关数据将被传输至并存储于该境内服务器。若您在中国境外使用，请知悉并同意上述跨境传输与存储。上传的图片等文件存储于阿里云对象存储（OSS）。

## 四、信息共享

我们**不会出售**您的个人信息。仅在以下情况共享：(1) 为实现 AI 对话功能，将必要的论文内容/提问发送至所选的大模型服务商；(2) 法律法规要求或为保护本应用与用户合法权益所必需。

## 五、第三方模型 API Key

若您在设置中填写自己的大模型 API Key，该 Key 仅用于代您发起对话请求；它存储在您的设备本地，请求时随本次对话发送至您指定的服务商地址。

## 六、您的权利

您可以随时在 App 内查看、修改您的资料与偏好，删除收藏/对话记录，或通过「删除账户」永久注销账号及相关数据。

## 七、未成年人

本应用面向科研人员，不面向未满 14 周岁的未成年人。

## 八、政策变更与联系方式

我们可能适时更新本政策，重大变更会在 App 内提示。如对本政策有任何疑问、意见或投诉，请联系：**support@aiforecon.cn**。
